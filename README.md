# 🎮 Rock Paper Scissors Game (C++)

A simple and interactive **Rock-Paper-Scissors** console game built using **C++**, where the player competes against the computer in multiple rounds.

> This project demonstrates fundamental C++ concepts such as enums, structs, functions, and control flow.
---

## 🚀 Features

* 🎯 Play multiple rounds (1 → 10)
* 🤖 Random computer choices
* 🧠 Smart round winner logic
* 📊 Final game statistics
* 🎨 Colored console output (Win / Lose / Draw)
* 🔁 Option to replay the game

---

## 🛠️ Technologies Used

* C++
* Standard Library (`iostream`, `cstdlib`, `ctime`)
* Console-based UI

## ⚙️ How to Run

### 1. Compile the code:

```bash
g++ main.cpp -o game
```

### 2. Run the game:

```bash
./game
```

## 🧠 Game Logic

* Stone beats Scissor
* Scissor beats Paper
* Paper beats Stone
* Same choice = Draw
